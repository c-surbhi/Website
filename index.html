Academic Projects

  • Applications of Survival Analysis on Orthopedic diseases (with R, Excel). (M.Sc. Sem- III)

    • Objective: Collection and Analysis of Data to find major causes behind commonly prevalent Orthopedic Diseases.

      • Steps involved:
      • Data Collection: On field data collection by conducting Survey from the target population.
      • Data Cleansing: Removing missing values, identifying and removing errors and inconsistent data in order to improve the quality of the data.
      • Data transformation and Data Visualization: Converting data into suitable form for further analysis and to extract useful variables and identifying relationships among variables.
      • Kaplan Meier Curve Analysis: Using survival Analysis tools such as Survival function, Kaplan Meier Curves, Hazard modelling, Log rank test, random forest for Survival Data (for feature selection).
      • Interpretation of the outputs: Interpret the results obtained and present the applicable suggestions in the context of real world that can help to solve the problem.
  • Machine learning applied to Survival Analysis with R. (M.Sc Sem- IV)

    • Objective: To compare various Survival models and find best one to predict the re-occurrence of orthopedic diseases.

      • Steps involved:
      • Data transformation and Data Visualization: Converting data into suitable form for further analysis and using various plots to identify trends, patterns.
      • Descriptive analysis: Removing missing values, identifying and removing errors and inconsistent data in order to improve the quality of the data. This will help to draw a clear picture of what information the data is providing to us.
      • Model Comparison: Apply 6 different Survival models with and compare them according to their performances and choose best model.
      • Variable Selection: Select the variables that are most suitable for model implementation to maintain the complexity and simplicity of the data.
      • Model Implementation: Split data into train and test set and apply model on the training data.
      • Testing the model accuracy: Use the test data to evaluate the prediciton power of the model in the sense how well the model is performing over the unseen data.
